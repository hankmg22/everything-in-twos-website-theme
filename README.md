# Two Tizm Website Theme

This repository holds the design and code for the [Two Tizm](https://twotizm.myshopify.com) online store. It controls how the site looks and behaves — the layout, colors, product pages, shopping cart, and all the little interactions shoppers experience.

## What's in here

- **The look of the site** — fonts, colors, spacing, and the overall layout of every page (home, product, cart, etc.)
- **The shopping cart** — adding items, changing quantities, and the slide-out cart drawer, all without the page needing to reload
- **Product pages** — size/option pickers, a size chart, and expandable info sections
- **Sound effects** — a small "click" sound plays when you tap buttons and links around the site
- **Photos and branding** — the logo, product photography, and other visual assets used throughout the store

## How it connects to the store

This isn't a standalone website — it's a *theme* that plugs into Shopify, the platform that runs the actual store, checkout, and order processing. Changes made to the files in this repo get pushed up to Shopify and show up live on twotizm.myshopify.com.

## Making changes

Any updates to how the site looks or works start here as file changes, then get pushed (uploaded) to the live Shopify store. If you're not comfortable editing the code yourself, that's totally fine — just describe what you'd like changed and it can be handled for you.
